About TeeRadar

We’re building a single doorway into Australian golf tee times. Instead of remembering each course’s website, golfers can browse once, filter once, and go straight to the right place.

This first version focuses on WA public and resort courses because many of them already expose dates in a similar format. As we collect usage, we’ll use that to request API access from the booking platforms.

What we do today

  • Show WA courses on a live map
  • Try to read availability from the course page
  • Send the golfer to the official booking page
  • Let phone-only courses appear with their number

What’s coming

  • State-by-state expansion
  • Better member profiles (home course, preferences)
  • Course-side integrations once approved
